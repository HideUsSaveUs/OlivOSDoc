# 插件

## 插件接口

### 发送消息

#### 说明
用于发送基本消息

| 参数 | 类型 | 解释 | 缺省 |
|:--:|:--:|:---|:--:|
| send_type | str | 用于指定发送目标的类型 | - |
| target_id | ID | 发送目标的ID | - |
| message | MSG | 用于指定发送目标的类型 | - |
| host_id  | ID | 发送目标的所属HOST ID | None |

- send_type

| 内容 | 解释 |
|:--:|:--:|:---|:--:|
| private | 私聊消息 |
| group | 群聊消息 |
